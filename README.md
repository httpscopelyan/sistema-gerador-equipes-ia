# sistema-gerador-equipes-ia
Gestor de Equipes Nissi é uma plataforma full stack com backend Node.js/TypeScript/Express, Sequelize e PostgreSQL, e frontend React/TypeScript/Vite. Processa planilhas de voluntários, integra com n8n/IA, gera equipes, salva workspaces e oferece dashboards, mapas e relatórios.
